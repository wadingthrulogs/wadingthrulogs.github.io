## Linux 

**Bandit** is a fun game that was made to help teach beginners Linux command line. You SSH into a host with the goal of obtaining the password for the next. The do give you some guides lines as well as the Commands you may need to solve the level. OverTheWire has several other wargames that are just as fun.

https://overthewire.org/wargames/bandit/
